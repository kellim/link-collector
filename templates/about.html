About Link Collector

Link Collector is a project for Udacity's Full Stack Web Developer Nanodegree created by Kelli McMillan. The code for Link Collector is on GitHub.

The assignment was to create an item catalog, so I chose links for my items. I have come across so many links related to different programming languages, frameworks, courses, etc, that I am always looking for ways to organize and share these links.

Part of the assignment included others being able to add and edit their own items, or in this case, links. This is great, because I'm always on the lookout for new and interesting links!
